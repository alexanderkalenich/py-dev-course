Программа обрабатывает данные по ценным бумагам из файлов CSV находящихся в директории 'trades', вычисляет 3 тикера с
максимальной и 3 тикера с минимальной волатильностью.
Бумаги с нулевой волатильностью выводятся отдельно.
Результаты выводятся на консоль в виде:
===========================
Максимальная волатильность:
       ТИКЕР1 - ХХХ.ХХ %
       ТИКЕР2 - ХХХ.ХХ %
       ТИКЕР3 - ХХХ.ХХ %
Минимальная волатильность:
       ТИКЕР4 - ХХХ.ХХ %
       ТИКЕР5 - ХХХ.ХХ %
       ТИКЕР6 - ХХХ.ХХ %
Нулевая волатильность:
       ТИКЕР7, ТИКЕР8, ТИКЕР9, ТИКЕР10, ТИКЕР11, ТИКЕР12
===========================
Для обработки данных запустите файл "01_volatility.py";
для обработки данных в МНОГОПОТОЧНОМ стиле, запустите файл "02_volatility_with_threads.py";
для обработки данных в МНОГОПРОЦЕССНОМ стиле, запустите файл "03_volatility_with_processes.py".
