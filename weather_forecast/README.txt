Программа для прогноза погоды с консольным интерфейсом.
Примеры запуска программы:
	python 01_weather.py 2021-07-01 2021-07-02 -a 
 	python 01_weather.py 2021-07-02 2021-07-03 -g
	python 01_weather.py 2021-07-02 2021-07-03 -a
        python 01_weather.py 2021-07-02 2021-07-03 -g
        python 01_weather.py 2021-07-02 2021-07-03 -c

Подробная информация указана в --help.
