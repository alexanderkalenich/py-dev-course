Программа для определения количества очков по упрощенной системе, с помощью пакета argparse.
# Скрипт принимает параметр --result и печатает на консоль:
# Количество очков для результатов ХХХ - УУУ.

Пример запуска консольной утилиты: python 01_score.py 43X-/211744XX32--
Пример полученного результата: "Количество очков для результата 43X-/211744XX32-- - 106"
